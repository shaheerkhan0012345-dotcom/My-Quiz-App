# My-Quiz-App
A simple and interactive JavaScript quiz application that allows users to answer multiple-choice questions and get instant scores. This project demonstrates fundamental JavaScript concepts such as DOM manipulation, event handling, and dynamic content rendering. Features include:

Display of multiple-choice questions

Tracking user score

Next question navigation

Final score display

Option to restart the quiz

Technologies Used:

HTML

CSS

JavaScript

Ideal For: Beginners who want to practice JavaScript, DOM manipulation, and building interactive web applications.
